# 1. 왜 자격증 교육 시장인가

## 시장 기회

한국에는 공인중개사, 주택관리사, 감정평가사 등 **노후 준비를 위해 취득하는 자격증** 시장이 있다. 이 시장의 핵심 특징:

| 특징 | 설명 |
|------|------|
| **타깃 연령** | 40~60대 (노후 준비, 제2의 커리어) |
| **수험 기간** | 6개월~3년 (장기 학습) |
| **학습 방식** | 교재 + 인강 + 기출문제 반복 풀이 |
| **핵심 페인포인트** | "해설을 봐도 이해가 안 됨", "왜 틀렸는지 모름", "법률 용어가 어려움" |

## 기존 플레이어의 한계

### 기존 기출 앱 (오르조, 뉴비티 등)

| 항목 | 현재 상태 |

# 2. 5개 자격증 중 왜 2개로 좁혔는가

## 검토한 5개 자격증

| 자격증 | 연간 응시자 | 잠재 시장 | 추세 | 2차 형식 |
|--------|-----------|----------|------|---------|
| 공인중개사 | ~50,000명 | ~550억원 | 급감 | **객관식** |
| 주택관리사 | ~19,000명 | ~155억원 | 안정 | 객관식+주관식 단답형 |
| 감정평가사 | ~8,600명 | ~190억원 | 증가 | **논문형 논술** |
| 행정사 | ~8,800명 | ~55억원 | 증가 | **서술형·약술형** |
| 관세사 | ~1,400명 | ~38억원 | 감소 | **논술형** |

## 탈락 이유

### 감정평가사 / 행정사 / 관세사 → 1단계에서 제외

| 제외 이유 | 설명 |
|----------|------|
| **논술 첨삭의 AI 품질 리스크** | 감정평가사 2차 공식 모범답안이 미공개. AI가 틀린 첨삭을 주면 신뢰를 한 번에 잃음. |
| **기술 복잡도 과도** | 논술 첨삭은 기존 AI 기반 틀(객관식 해설 엔진)로 구현 어려움. 별도의 전문가 검수 프로세스 필요 → 초기 스타트업에 과도한 비용. |
| **사용자 볼륨 부족** | 감정평가사 8,600명, 행정사 8,800명, 관세사 1,400명 — 앱 초기 성장에 불리. |

### 공인중개사 + 주택관리사 → 최종 선정

| 선정 이유 | 공인중개사 | 주택관리사 |
|----------|----------|----------|
| **시험 형식** | 1·2차 모두 **객관식** → AI 해설 엔진 하나로 커버 | 1차 객관식 + 2차 객관식/단답형 → 동일 엔진 활용 |
| **사용자 규모** | ~50,000명 (줄고 있지만 아직 최대 볼륨) | ~19,000명 (안정적) |
| **타깃 일치** | 40~60대 노후 준비 수험생 | 40~60대 노후 준비 수험생 |
| **기존 앱 해설의 빈약함** | 오르조 등 1줄 해설만 제공 | 기출 앱 자체가 적음 |
| **교재 DB 공유** | 민법(1차)이 주택관리사와 겹침 → **코어 엔진 공유** | 민법(1차), 법규 체계 유사 |
| **합산 볼륨** | **약 7만명** (공인중개사 5만 + 주택관리사 1.9만) | |

# 4. 우리가 만드는 것

## 제품 정의

> **교재 DB 기반 AI 맥락 해설 앱**
> 기출문제를 풀고, 각 문제마다 AI가 "왜 이 선지가 맞고, 왜 저 선지가 틀린지"를 교재 내용에 근거하여 자세하게 설명해주는 앱.

## 기존 앱과의 차이

| | 기존 기출 앱 (오르조, 뉴비티 등) | **우리 앱** |
|--|------|------|
| 해설 방식 | 정답/오답 1줄 (고정 텍스트) | 교재 DB 기반 **맥락적 해설** |
| "왜 틀렸는지" | 거의 없음 | **오답 선지별로 근거 제시** ("민법 제XX조에 의하면...") |
| 헷갈리는 부분 | 무시됨 | **수험생들이 자주 혼동하는 포인트를 선제적으로 짚어줌** |
| 관련 개념 연결 | 없음 | "이 개념은 2019년 XX회에도 출제됨" / "OO법 제X조와 비교" |
| 추가 질문 | 불가 | **"이 부분 더 설명해줘"** 대화형 |
| 난이도 맞춤 | 없음 | **수험생 수준에 맞는 쉬운 설명** (40~60대 타깃) |
| 약점 분석 | 없음 | 틀린 문제 패턴 → **취약 과목/단원 자동 추천** |

## 왜 이것이 먹히는가

1. **타깃이 40~60대**: 강의를 반복 듣기 어렵고, 물어볼 데가 없음. "내 페이스대로 물어보면 답해주는" AI가 강의보다 편할 수 있음.
2. **법률 용어 장벽**: 공인중개사·주택관리사 모두 민법, 부동산법, 세법 등 법률 과목 비중이 높음. "해설을 봐도 이해 안 됨"이 최대 불만. AI가 쉬운 말로 풀어줌.
3. **기출 반복이 합격의 핵심 전략**: 두 자격증 모두 "10년 기출 반복"이 공통 학습법. 기출 풀이 + 깊은 해설의 조합이 가장 큰 가치.
