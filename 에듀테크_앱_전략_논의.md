# 에듀테크 앱 전략 논의 기록

> 날짜: 2026-03-25
> 목적: 자격증 교육 시장을 타깃으로 한 에듀테크 앱 기획

---

## 1. 프로젝트 배경

한국에는 주택관리사, 관세사, 감정평가사 등 다양한 자격증이 있으며, 주로 **노후 준비를 하는 사람들이 새로운 도전**을 위해 취득하는 형태의 자격증이다. 이 자격증 교육 시장을 타깃으로 한 **에듀테크 앱**을 만들고자 한다.

### 타깃 자격증 (스크린샷 기반)
- 공인중개사
- 주택관리사
- 행정사
- 감정평가사
- 관세사
- 금융·회계·무역 계열
- 디자인/AI/개발 계열

## 3. 앱 컨셉

### 핵심 기능
- **DB에 모든 교재 데이터를 정리된 형태로 입력**
- 논술문제든 기출이든 **아주 정확하고 세세하게 AI가 해설**해주는 앱

### 기존 기출앱과의 차이점

| | 기존 기출앱 (오르조, 뉴비티 등) | 만들려는 AI 해설 앱 |
|--|---------------------------|-------------------|
| 해설 방식 | 정답/오답 1줄 해설 (고정 텍스트) | 교재 DB 기반 **맥락적 해설** |
| "왜 틀렸는지" | 거의 없음 | **오답 선지별로 근거 제시** |
| 관련 개념 연결 | 없음 | "민법 제XX조 연결, 2019년에도 출제" |
| 추가 질문 | 불가 | **"이 부분 더 설명해줘"** 대화형 |
| 개인화 | 없음 | 틀린 문제 패턴 → **약점 자동 추천** |

## 4. 시장 진입 전략 논의

### 초기 검토: 감정평가사 1순위 추천

**감정평가사를 1순위로 추천한 이유:**

1. **페인포인트가 가장 큼** - 2차 논문형 시험의 답안 첨삭이 합격 핵심인데, 학원에만 의존 (하우패스 148~298만원)
2. **시장 성장 중** - 2025년 출원자 +1,859명 급증 (공인중개사는 9.3만→3.3만 급감)
3. **AI 해설 차별화가 가장 강함** - 논술형 모범답안 생성 + 첨삭은 기존 앱에 없는 영역
4. **교재 DB 구축 실현 가능** - 출판사 한정적, 법령/판례 기반이라 AI 환각 리스크 낮음
5. **LTV 높음** - 수험기간 2~3년, 합격률 7.31%로 재수험 비율 높음

### 재검토: 주택관리사의 재평가

**"객관식이라도 AI 해설 엔진은 다른 영역"이라는 반론이 제기됨.**

주택관리사가 재평가된 이유:
1. **법/회계 용어가 어려움** → "해설을 봐도 이해 안 됨"이 최대 불만 → AI가 수준 맞춤 설명
2. **응시자 19,000명** → 감정평가사의 2배 → 초기 사용자 확보 유리
3. **40~60대 타깃** → "물어보면 답해주는" 대화형 AI가 오히려 강의보다 편할 수 있음 (본인 페이스, 반복 질문 가능)

### 최종 전략: 동시 진입 + 단계적 확장

| 단계 | 시장 | 역할 |
|------|------|------|
| **1단계 동시** | **감정평가사** | 논술 첨삭으로 차별화 + 수익성 (객단가 200만+) |
| **1단계 동시** | **주택관리사** | AI 해설로 차별화 + 사용자 규모 (19,000명) |
| 2단계 | 행정사 | 논술형 확장 + 성장세 |
| 3단계 | 공인중개사 | 볼륨 시장, AI 해설로 기존 앱과 차별화 |
| 4단계 | 관세사 | 고객단가 높은 니치 시장 |

**핵심 로직:** 감정평가사는 **수익성과 차별화**(논술 첨삭), 주택관리사는 **사용자 규모**(19,000명)로 역할을 나누고, 교재 DB + AI 해설 엔진이라는 코어는 동일하게 공유.

## 5. 앱으로 대체 가능한 학습 영역 분석

| 학습 방법 | 현재 상태 | 앱 대체 가능성 |
|----------|----------|--------------|
| 기출문제 풀이 | 무료 앱 다수 (오르조 120만 유저) | 높음 - 단, **AI 해설 깊이**로 차별화 |
| 요약노트/암기카드 | 수험생이 직접 제작 | **매우 높음** - AI 자동 생성 |
| 스터디그룹 매칭 | 네이버/Daum 카페 수동 모집 | 높음 |
| 모의고사/성적 분석 | 학원 오프라인/PDF | 높음 - 실시간 취약점 분석 |
| 답안 첨삭 (논술형) | 학원 의존도 매우 높음 | **중~높음** - AI 첨삭 |
| 기본이론 강의 | 유튜브 무료 + 유료 인강 | 낮음 - 레드오션 |

## 6. 핵심 인사이트 정리

1. **단순 기출 앱은 레드오션** (오르조 120만 사용자) → "교재 DB 기반 AI 맥락 해설"이 차별화 포인트
2. **논술형 자격증(감정평가사/행정사/관세사)에서 AI 첨삭은 블루오션** → 현재 학원만 가능
3. **객관식이라도 AI 해설의 깊이는 완전히 다른 제품** → 주택관리사/공인중개사에서도 경쟁력 확보 가능
4. **에듀윌 매출 급감**(1,557→826억)이 보여주듯 공인중개사 단일 의존은 리스크
5. **메가랜드 완전자본잠식** → 자격증 교육 후발 진입은 기존 방식으로는 불가, **AI/앱이라는 다른 접근 필수**
6. **노후 준비 수험생(40~60대)** → 대화형 AI가 오히려 강점 (자기 페이스, 반복 질문, 부담 없음)
