## 기출문제 앱 시장의 공통 한계점 (= 사업 기회)

1. **해설이 간략함** — 정답/오답 1줄 해설이 대부분. "왜 틀렸는지" 맥락적 설명 부재
2. **2차(서술형/논술형) 대비 기능 전무** — 기존 앱은 모두 1차 객관식에만 특화
3. **AI 기능 없음** — 오르조 120만 유저지만 AI 관련 기능 거의 없음
4. **추가 질문 불가** — 고정 텍스트 해설, 대화형 학습 불가
5. **개인화 없음** — 약점 자동 추천, 수준 맞춤 설명 미제공
