## 공인중개사

### 1) 독학 (교재만)
- 독학 합격 가능하지만, 최소 **6개월~1년** 이상 필요
- 1년 계획을 월/주/일별로 세분화하여 관리
- 암기보다 **내용 이해 → 반복 읽기** 전략이 효과적
- **본인이 직접 요약본을 만드는 것**을 권장 (남의 요약본은 비추)
- 독학 3개월 만에 포기하는 사례도 많아 자기 관리가 핵심

## 행정사

### 1) 독학 가능성
- **1차 시험은 독학 가능**, 2차 논술형은 피드백/사례 연습 필수
- 직장 병행: 평일 2시간, 주말 6시간으로 꾸준히 공부 → 합격 사례 있음
- **기출문제집 반복**이 핵심 (기출된 주제만 공부해도 합격 가능)
- 기본서 2~3회 반복 정독 → 기출문제로 출제 경향 파악
- 합격자 사례: "기출문제를 매일 한 문제씩 답안지에 작성하여 합격"

# Part 3: 에듀테크 앱 관점 종합 시사점

## 앱으로 대체 가능한 학습 영역

| 학습 방법 | 현재 상태 | 앱 대체 가능성 |
|----------|----------|--------------|
| **기출문제 풀이** | 무료 앱 다수 (오르조 120만, 뉴비티 등) | 높음 - 단, 차별화 필요 |
| **요약노트/암기카드** | 수험생이 직접 만드는 경우 대부분 | **매우 높음** - AI 자동 생성 가능 |
| **스터디그룹 매칭** | 네이버/Daum 카페에서 수동 모집 | **높음** - 온라인 매칭 플랫폼 |
| **모의고사/성적 분석** | 학원 오프라인 또는 PDF 배포 | **높음** - 실시간 취약점 분석 |
| **답안 첨삭** (감평/관세/행정 2차) | 학원 의존도 매우 높음 | **중간** - AI 첨삭 가능하나 신뢰도 과제 |
| **기본이론 강의** | 유튜브 무료 + 유료 인강 레드오션 | 낮음 - 기존 업체 강점 |
| **커뮤니티** | 공사모(21만), 부동산스터디(163만) 기존 강세 | 낮음 - 이탈 유도 어려움 |

## 핵심 인사이트

1. **기출문제 앱 시장은 이미 경쟁 치열** (오르조 120만 사용자) → 단순 기출 앱으로는 차별화 어려움
2. **감정평가사/관세사/행정사 2차의 "답안 첨삭"이 최대 페인포인트** → AI 논술 첨삭이 차별화 포인트
3. **노후 준비 수험생 타깃**은 디지털 리터러시가 낮을 수 있어 **UI/UX 단순화** 필수
4. **에듀윌 매출 급감**(1,557억→826억)이 보여주듯, 공인중개사 단일 의존은 리스크
5. **행정사/감정평가사는 성장세** → 이쪽 선점이 전략적
6. **메가랜드의 완전자본잠식**이 보여주듯, 자격증 교육 시장 후발 진입은 매우 어려움 → 기존과 다른 접근(앱/AI) 필수
