# AI 기출해설 앱 — 최종 제안서

> 작성일: 2026-03-26

---

# 1. TLDR

**공인중개사 + 주택관리사** 자격증 수험생을 위한 **AI 기출문제 해설 앱**을 만든다.

기존 기출 앱(오르조, 뉴비티 등)은 "정답 1줄 해설"에 그치지만, 우리 앱은 **모든 선지에 대해 "왜 맞고 왜 틀린지"를 AI가 맥락적으로 설명**하고, 이해가 안 되면 **추가 질문**할 수 있는 대화형 AI 튜터.

타깃은 **40~60대 노후 준비 수험생**. 이들은 법률 용어가 어렵고, 해설을 봐도 이해가 안 되며, 물어볼 곳이 없어 포기한다. 이 문제를 AI로 해결한다.

3줄 요약 하자면,

- **기존 기출 앱의 인프라 부실**
- **실제로 기출 해설 부족이라는 'pain point' 가 존재하고, 여기에 AI 해설이 들어가면 수요가 높을 것으로 예상**
- **타깃 나이 및 시장 규모가 기본적으로 우즈벡 TOPIK 시장의 약 15배 규모. 신규 진입 시장으로 적절**

# 2. 공인중개사, 주택관리사가 뭔가

## 공인중개사

부동산 거래를 중개하는 전문 자격증. 한국에서 가장 많이 응시하는 자격증 중 하나.

| 항목 | 내용 |
|------|------|
| 연간 응시자 | ~50,000명 (2024 기준, 정점 대비 감소 중) |
| 시험 형식 | **1·2차 모두 객관식 5지선다** |
| 과목 | 1차: 부동산학개론, 민법 / 2차: 중개사법, 부동산공법, 공시법·세법 |
| 합격률 | 1차 15.08% (2024), 2차는 이보다 높음 |
| 수험 기간 | 보통 6개월~1년 |
| 주요 수험생 | **40~60대**, 노후 준비·제2의 커리어 목적 |

- 출처: [Q-Net 공인중개사](https://www.q-net.or.kr/crf005.do?gId=08&gSite=L&id=crf00503)
- 출처: [2024 제35회 시험 결과 통계](https://dinoedu.co.kr/15)

## 주택관리사 (주택관리사보)

아파트 등 공동주택을 관리하는 전문 자격증. 관리소장이 되기 위한 필수 자격.

| 항목 | 내용 |
|------|------|
| 연간 응시자 | ~19,000명 (안정적) |
| 시험 형식 | 1차 객관식 / 2차 **객관식 + 주관식 단답형** |
| 과목 | 1차: 민법, 회계원리, 시설개론 / 2차: 관계법규, 관리실무 |
| 수험 기간 | 비전공자 기준 1~2년 |
| 주요 수험생 | **40~60대**, 노후 안정적 직업 목적 |

- 출처: [Q-Net 주택관리사](https://www.q-net.or.kr/crf005.do?id=crf00503&jmCd=9744)

# 3. 왜 이 2개인가

## 5개 후보에서 2개로 좁힌 이유

처음에 검토한 자격증은 5개였다.

| 자격증 | 응시자 | 2차 형식 | 선정 여부 |
|--------|--------|---------|----------|
| **공인중개사** | ~50,000 | **객관식** | **✅ 선정** |
| **주택관리사** | ~19,000 | 객관식+단답형 | **✅ 선정** |
| 감정평가사 | ~8,600 | 논문형 논술 | ❌ 제외 |
| 행정사 | ~8,800 | 서술형·약술형 | ❌ 제외 |
| 관세사 | ~1,400 | 논술형 | ❌ 제외 |

## 객단가는 높으나 감정평가사·행정사·관세사를 제외한 이유

| 이유                   | 설명                                                 |
| -------------------- | -------------------------------------------------- |
| **논술 첨삭의 AI 품질 리스크** | 감정평가사 2차는 공식 모범답안 미공개. AI가 틀린 첨삭을 주면 신뢰를 한 번에 잃음   |
| **기술 복잡도 과도**        | 논술 첨삭은 객관식 해설 엔진과 완전히 다른 기술. 초기 스타트업에 과도한 비용       |
| **사용자 볼륨 부족**        | 감정평가사 8,600명, 행정사 8,800명, 관세사 1,400명 — 앱 초기 성장에 불리 |

## 공인중개사 + 주택관리사를 선정한 이유

**① 타깃이 동일하다**
- 둘 다 **40~60대 노후 준비** 수험생이 핵심
- "물어보면 답해주는" AI가 강의보다 편한 세대

**② 시험 형식이 AI 해설에 최적화되어 있다**
- 공인중개사: 1·2차 **전부 객관식** → AI 해설 엔진 하나로 완벽 커버
- 주택관리사: 1차 객관식 + 2차 객관식/단답형 → 동일 엔진 활용 가능

**③ 합산 볼륨이 가장 크다**
- 연간 응시자 합산 **약 7만명**, 준비생 포함 시 **약 10만명**
- 나머지 3개 자격증 전부 합쳐도 약 1.9만명

**④ 코어 엔진을 공유할 수 있다**
- 두 시험 모두 **민법**이 1차 과목 → 같은 DB·해설 엔진 재사용
- 부동산 관련 법규 체계가 유사

# 7. 어떻게 만들 것인가

## 앱 핵심 기능

### ① 기출문제 + AI 전체 선지 해설

문제를 풀면, AI가 **5개 선지 모두**에 대해 왜 맞고 왜 틀린지 해설.

| | 기존 앱 | 우리 앱 |
|--|--------|--------|
| ①번 (정답) | "정답" 표시만 | **"민법 제107조 1항에 의하면... 쉽게 말해 ○○라는 뜻입니다"** |
| ②번 | 표시 없음 | **"이 선지는 △△를 말하고 있는데, 실제로는 ☆☆이므로 틀립니다"** |
| ③번 | 표시 없음 | **"이 선지가 함정입니다. ①번과 헷갈리기 쉬운데, 차이점은..."** |
| ... | | |

### ② 대화형 추가 질문

해설을 봐도 모르겠으면 **"이 부분 더 쉽게 설명해줘"** → AI가 일상 언어로 재설명.

```
사용자: "비진의 표시가 뭔데?"
AI: "쉽게 말하면, 농담처럼 진심이 아닌 말을 한 건데
     상대방이 진심으로 받아들인 상황이에요.
     예를 들어, '내 집 100원에 줄게'라고 했는데
     상대가 진짜 계약서를 쓴 경우..."
```

### ③ 약점 자동 분석

틀린 문제의 패턴을 분석하여 **"민법 물권 파트가 약합니다. 이 단원 기출 10문제를 더 풀어보세요"** 추천.

## 기술 구조

| 구성 요소        | 소스                              | 저작권     |
| ------------ | ------------------------------- | ------- |
| **기출문제**     | Q-Net 공식 공개 기출                  | ✅ 문제 없음 |
| **정답 데이터**   | Q-Net 공개 + 검증                   | ✅ 문제 없음 |
| **AI 해설 엔진** | AI 모델(GPT-4/Claude) + 법령 원문 RAG | ✅ 문제 없음 |
| **법령 DB**    | 국가법령정보센터 공개 법령·판례               | ✅ 문제 없음 |

> **교재 PDF는 사용하지 않는다.** AI 모델이 이미 법률 개념을 알고 있고, 법령 원문을 RAG로 참조하면서 해설을 생성한다. AI가 1차 해설을 만들고, 전문가가 검수하는 구조.

# 9. 확장 시나리오: 교재 DB를 추가한다면

> 기본 구조는 법령 RAG + AI 해설이지만, **교재 데이터를 DB에 추가할 수 있다면** 해설 품질과 사용자 가치가 한 단계 올라간다.

## 왜 교재 DB가 있으면 더 좋은가

| | 법령 RAG만 (기본) | 법령 RAG + 교재 DB (확장) |
|--|-----------------|----------------------|
| 해설 근거 | "민법 제107조에 의하면..." | **"에듀윌 기본서 p.234에서 설명하듯, 민법 제107조는..."** |
| 개념 설명 | AI가 일반적 설명 생성 | **교재의 설명 방식·예시를 그대로 활용** → 수험생이 교재와 연결해서 복습 가능 |
| 출제 포인트 | AI 추론에 의존 | **교재에서 강조한 부분과 기출의 매칭** → "이 부분은 에듀윌 요약집에서 ★표시된 핵심 포인트입니다" |
| 학습 동선 | 기출 → AI 해설 → 끝 | 기출 → AI 해설 → **"교재 OO페이지에서 이 개념을 더 공부하세요"** → 교재로 돌아감 |

**핵심**: 수험생은 이미 교재를 갖고 있다. AI 해설이 **"교재의 어디를 보면 되는지"**까지 알려주면, 앱과 교재가 연동된 학습 경험이 된다.

## 교재 DB 추가 시 저작권 이슈 및 해결 방안

| 방식 | 저작권 리스크 | 실현 가능성 |
|------|------------|-----------|
| **교재 PDF 통째로 DB화** | ❌ 높음 — 출판사 저작권 침해 | 비추천 |
| **교재 내용을 AI 학습 데이터로 사용** | ⚠️ 중간 — 저작권법 해석에 따라 다름 | 법률 검토 필요 |
| **출판사와 공식 제휴** | ✅ 없음 — 라이선스 계약 체결 | **추천. 에듀윌/경록과 제휴하여 교재 데이터 라이선스 확보** |
| **교재 목차·페이지 참조만** (내용 미포함) | ✅ 없음 — 사실 정보 참조 | 추천. "에듀윌 기본서 p.234 참조"까지만 안내 |
| **사용자가 자기 교재를 스캔/업로드** | ⚠️ 낮음 — 개인 사용 범위 | 가능. 사용자가 구매한 교재를 본인용으로 업로드 |

## 추천 단계별 접근

| 단계             | 교재 활용 방식                                                  |
| -------------- | --------------------------------------------------------- |
| **MVP (출시 시)** | 교재 없이 법령 RAG만. "민법 제XX조에 의하면..." 수준                       |
| **v1.5**       | **교재 목차·페이지 참조** 추가. "이 개념은 에듀윌 기본서 3장에서 다룹니다"            |
| **v2.0**       | **출판사 제휴** 체결 후 교재 내용 기반 해설. "에듀윌 기본서에서 설명하듯, 비진의 표시란..." |


> **v2.0의 출판사 제휴가 핵심 마일스톤.** 에듀윌(공인중개사 교재 1위)과 제휴하면, "에듀윌 교재를 쓰는 수험생은 우리 앱을 쓰면 교재와 연동된 AI 해설을 받을 수 있다"는 강력한 유입 경로가 된다. 에듀윌 입장에서도 교재 판매 촉진 효과가 있어 윈윈 구조.

# 10. 기존 앱(우즈베키스탄 한국어시험) 시장과의 비교

> 기존에 운영하던 앱이 우즈베키스탄 한국어검정시험(TOPIK) 시장이었다. 이 시장과 한국 자격증 시장을 비교하여 전환의 근거를 정리한다.

## 두 시장 핵심 비교

| 비교 항목             | 우즈베키스탄 한국어시험            | 한국 자격증 (공인중개사+주택관리사)     |
| ----------------- | ----------------------- | ------------------------ |
| **타깃 사용자 수**      | ~20,000명                | **~100,000명 (5배)**       |
| **1인당 구매력 (GDP)** | $2,255                  | **$33,000+ (15배)**       |
| **시장 성장성**        | 높음 (글로벌 한국어 CAGR 25.1%) | 혼합 (공인중개사 감소, 행정사/감평 증가) |
| **경쟁 강도**         | 낮음 (앱 경쟁자 적음)           | 중간 (오르조 120만, 기존 학원)     |
| **확장성**           | 중앙아시아 (카자흐, 키르기스 등)     | 다른 자격증 (감정평가사, 행정사 등)    |

## 전환 근거

**한국 자격증 시장이 약 10배 이상 큰 시장이다.**

"사용자 수 5배 × 구매력 15배"의 곱이 결정적 차이. 같은 기술(AI 해설 엔진)을 만들더라도 한국 자격증 시장에 적용했을 때 수익이 11배 높다.

다만 우즈베키스탄 시장은 **글로벌 한국어 학습 붐**(2024년 72억달러 → 2034년 670억달러, CAGR 25.1%)의 혜택을 받을 수 있어, 장기적으로 부수적 수익원으로 유지하면서 한국 자격증 시장을 메인으로 가져가는 것이 합리적.
