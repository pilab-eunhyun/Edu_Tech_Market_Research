## 3. 직업/배경

> 공식 통계에서 직업별 분류를 상세 공개하지 않으므로, 교육업체 설문 및 언론 보도 기반으로 정리.

### 주요 응시층

| 유형 | 설명 |
|------|------|
| **직장인 (투잡/전직 목적)** | 현직 직장인이 퇴직 후를 대비하거나 부수입 목적으로 응시 |
| **주부/경력단절 여성** | 자녀 성장 후 재취업 목적. 중개사무소 창업이 매력적 |
| **퇴직자/은퇴 준비자** | 50대 이상, 노후 대비 제2의 직업 준비 |
| **자영업자** | 부동산 관련 지식 습득 또는 업종 전환 목적 |
| **대학생/취준생** | 취업 스펙 또는 부동산 분야 진출 목적 (비율 낮음) |

### 40~50대 자격증 선호도 순위

| 순위 | 40대 | 50대 |
|------|------|------|
| 1위 | **공인중개사** | **공인중개사** |
| 2위 | 주택관리사 | 주택관리사 |
| 3위 | 사회복지사 1급 | 전기기사 |
| 4위 | 전기기사 | 사회복지사 1급 |
| 5위 | 전산세무회계 | 산업·건설안전기사 |

**→ 40대·50대 모두 공인중개사가 가장 인기 있는 자격증**

## 4. 수험 동기

### 주요 동기

| 동기 | 설명 |
|------|------|
| **노후 준비 / 제2의 직업** | 가장 핵심적인 동기. "은퇴 후에도 일할 수 있는 직업" |
| **창업 (중개사무소 개업)** | 자격증만으로 바로 개업 가능 → "나만의 평생직장" |
| **전직/이직** | 현재 직장에서의 경력 전환 수단 |

| **부동산 투자 지식** | 본인 투자를 위한 전문 지식 습득 목적 |

## 5. 소득 수준

> 공인중개사 응시자의 소득에 대한 직접적인 공식 통계는 없음. 간접 데이터로 추정.

### 수험 비용

| 항목 | 비용 |
|------|------|
| 인강 수강료 | 30~80만원 |
| 교재비 | 10~20만원 |
| 에듀윌 평균 결제 단가 | 약 50~60만원 |
| 시험 응시료 | 1차 13,000원, 2차 14,000원 |

## 6. 학력 분포

> **공식 통계 없음** — 공인중개사 시험은 학력 제한이 없어 별도의 학력별 통계가 공개되지 않음.

- 응시 자격에 학력 요건이 없으므로 고졸부터 대학원 이상까지 다양한 학력의 응시자 존재
- 40~50대가 주 응시층이므로 해당 세대의 일반적 학력 분포(고졸~대졸)가 반영될 것으로 추정 [추정]
