## 1-2. 기출문제의 실제 구조

### 문제 유형 분류

공인중개사 기출문제는 크게 4가지 유형으로 분류된다:

| 유형 | 형태 | 비율 (추정) | 예시 |
|------|------|-----------|------|
| **정오 판단형** | "다음 중 옳은 것은?" / "틀린 것은?" | ~50% | "민법상 비진의 표시에 관한 설명으로 옳은 것은?" |
| **개수 판단형** | "옳은 것을 모두 고른 것은?" / "몇 개인가?" | ~20% | "다음 중 옳은 것을 모두 고른 것은? (ㄱ, ㄴ, ㄷ, ㄹ)" |
| **사례 적용형** | 사례 제시 후 법적 판단 | ~15% | "甲이 乙에게 토지를 매도한 경우..." |
| **계산형** | 수치 계산 (투자론, 세법 등) | ~15% | "순영업소득이 5,000만원이고 환원이율이 10%일 때 부동산 가치는?" |

**최근 출제 트렌드:**
- "옳은 것(또는 틀린 것)을 **모두** 골라 몇 개인가?" 유형이 증가 → **5개 선지 전체를 정확히 판단해야 풀 수 있는 문제**
- 이는 "1줄 해설"로는 학습이 안 되는 근본적 이유

- 출처: [브런치 - 공인중개사 10년 기출문제 완벽분석](https://brunch.co.kr/@richboy/488)

### 실제 문제 형태 예시

```
[제35회 공인중개사 1차 - 민법 및 민사특별법]

문 3. 민법상 비진의 의사표시에 관한 설명으로 옳은 것은?

① 의사표시는 표의자가 진의 아닌 것을 알고 한 것이라도 그 효력이 있다.
② 비진의 의사표시에서 상대방이 표의자의 진의 아님을 알았거나 알 수 있었을
   경우에도 의사표시는 유효하다.
③ 비진의 의사표시의 무효는 선의의 제3자에게도 대항할 수 있다.
④ 비진의 의사표시에서 상대방이 표의자의 진의 아님을 알았을 경우,
   그 의사표시는 취소할 수 있다.
⑤ 비진의 의사표시는 표의자와 상대방이 통정한 경우에만 무효이다.

정답: ①
```

**→ 핵심: 5개 선지 각각이 독립된 법률 지식을 테스트함. 정답이 왜 맞는지뿐 아니라 나머지 4개가 왜 틀린지를 아는 것이 학습의 본질.**

## 1-3. 현재 해설의 문제점 — 수험생이 겪는 구체적 불편

### 기출문제집의 해설 방식 비교

디시인사이드 공인중개사 갤러리(조회 3,334, 추천 31)에서 지적된 핵심 문제:

| 비교 항목 | 공무원 시험 기출문제집 | **공인중개사 기출문제집** |
|----------|---------------------|----------------------|
| 정답 선지 해설 | ✅ 상세 해설 + 이론 확장 | ✅ 해설 있음 |
| **오답 선지 해설** | ✅ **틀린 지문도 전부 해설** + 관련 이론 확장 | ❌ **옳은 지문은 절대 해설 안 달아줌** |
| 독학 가능 여부 | 기출문제집만으로 독학 가능 | 기출문제집만으로 독학 어려움 |

> **"공인중개사 기출문제집은 옳은 지문에 대해서는 해설을 달지 않는다. 공무원 기출집은 모든 선지를 해설하고 이론까지 확장하는데, 공인중개사 기출집은 안 한다."**
> — 디시인사이드 공인중개사 갤러리

**→ 이것이 우리 앱의 핵심 기회: 5개 선지 전체에 대한 AI 해설**

## 1-4. 수험생의 실제 학습 워크플로우

### 일반적 학습 단계 (합격자 패턴)

```
1단계: 기본서 정독 (약 2~3개월)
  └→ 에듀윌/경록/박문각 기본서 5권 1회독

2단계: 기출문제 풀이 + 해설 학습 (약 3~4개월, 핵심 단계)
  └→ 기출문제집으로 10년치 풀기
  └→ 틀린 문제 기본서에서 해당 부분 발췌독
  └→ ★ 이 단계에서 "해설 부실" 문제 발생 ★
  └→ 모르는 부분 → 네이버 카페/에듀윌 지식iN에서 검색 (30분+ 소요)

3단계: 기출 반복 회독 (약 2~3개월)
  └→ 같은 기출을 3~5회 반복
  └→ "5개 선지의 옳고 그른 이유를 설명할 수 있으면 합격"
  └→ ★ 이 단계에서 "물어볼 곳 없음" 문제 발생 ★

4단계: 모의고사 + 약점 보완 (시험 직전 1~2개월)
  └→ 실전 모의고사
  └→ 계속 틀리는 문제 오답노트 작성
```

### 핵심 페인포인트가 발생하는 지점

| 학습 단계 | 발생하는 문제 | 현재 해결 방법 | 소요 시간 |
|----------|------------|-------------|----------|
| **기출 풀이 → 해설 학습** | 정답은 알겠는데 **왜 나머지 4개가 틀린지 모름** | 기본서 발췌독 + 검색 | 문제당 10~30분 |
| **오답 분석** | 해설이 "제107조 참조"로만 끝남 → **법조문을 봐도 이해 안 됨** | 네이버/카페 질문 (답변 대기 수시간~일) | 답변까지 수시간 |
| **개념 연결** | 비슷한 개념이 헷갈림 (예: 비진의 vs 통정허위) → **차이를 설명해주는 곳 없음** | 강의 다시 듣기 / 포기 | 30분~1시간 |
| **법 개정 혼란** | 최신 법령과 기출 당시 법령이 다름 → **어느 기준으로 답을 해야 하는지 혼란** | 스스로 확인 불가능 | — |

**→ AI 앱이 해결해야 할 것: 문제 풀이 직후 "5개 선지 전체 해설 + 추가 질문 + 헷갈리는 개념 비교"를 즉시 제공**

### AI 해설이 생성해야 할 내용 (선지당)

```
[선지 ①] "의사표시는 표의자가 진의 아닌 것을 알고 한 것이라도 그 효력이 있다."

→ [AI 해설]
   ✅ 정답입니다.

   📌 근거: 민법 제107조 제1항
   "의사표시는 표의자가 진의 아님을 알고 한 것이라도 그 효력이 있다."

   💡 쉬운 설명:
   "비진의 표시"란 농담처럼 진심이 아닌 말을 한 경우입니다.
   예를 들어, "내 집 100원에 줄게"라고 했는데 상대방이 진짜로 받아들인 경우,
   원칙적으로는 이 의사표시가 유효합니다.

   단, 상대방이 "이 사람이 진심이 아니구나"라고 알았거나 알 수 있었으면
   무효가 됩니다 (제107조 제1항 단서).

   ⚠️ 헷갈리기 쉬운 포인트:
   비진의 표시(제107조) vs 통정허위표시(제108조)의 차이:
   - 비진의: 혼자 속으로 "진심 아닌데" → 원칙 유효
   - 통정허위: 상대방과 짜고 "가짜로 하자" → 무효
```

**→ 이 수준의 해설이 5개 선지 전체에 대해 제공되어야 함.**

### 출제 패턴 분석 데이터 (추가 가치)

| 데이터 | 활용 방법 |
|--------|---------|
| 과목별·단원별 출제 빈도 | "이 단원에서 매년 3~4문제 출제" → 학습 우선순위 추천 |
| 선지 재활용 패턴 | "작년 ①번 선지가 올해 ③번으로 재출제" → 변형 문제 예측 |

| 함정 선지 패턴 | "①번과 ③번이 자주 헷갈리는 조합" → 비교 해설 자동 생성 |

## 2-3. 수험생의 실제 학습 워크플로우

### 학습 단계

```
1단계: 기본서 정독 (약 3~4개월)
  └→ 에듀윌/해커스 기본서 5권 (민법, 회계원리, 시설개론, 관계법규, 관리실무)
  └→ ★ 회계원리·민법에서 초기 이탈 발생 ("과락만 면하자" 수준)

2단계: 1차 기출문제 풀이 (약 2~3개월)
  └→ 객관식은 공인중개사와 동일한 패턴
  └→ ★ 동일한 "해설 부실" 문제 발생

3단계: 2차 대비 — 주관식 단답형 학습 (약 2~3개월, 별도 준비)
  └→ 법령 키워드 암기가 핵심
  └→ "직접 써보지 않으면 시험장에서 못 씀" — 필기 연습 필수
  └→ ★ 키워드를 정확하게 외웠는지 확인할 방법이 없음 ★
  └→ ★ 비슷한 용어 혼동 (예: "관리주체" vs "관리사무소장" vs "입주자대표회의") ★

4단계: 모의고사 + 약점 보완 (시험 직전)
```

### 핵심 페인포인트

| 학습 단계 | 발생하는 문제 | AI가 해결할 수 있는 방법 |
|----------|------------|---------------------|
| 1차 객관식 | 해설 부실 (공인중개사와 동일) | **5개 선지 전체 AI 해설** |
| 2차 주관식 | 키워드를 정확하게 암기했는지 확인 불가 | **빈칸 채우기 + AI가 정답 확인 + 유사 답안 부분 인정** |
| 2차 주관식 | 비슷한 법령 용어 혼동 | **AI가 "OO vs XX의 차이점" 비교 설명** |
| 회계원리 | 개념 자체가 어려움 (비전공자) | **회계 용어를 일상 언어로 재설명** |

## 기출문제 출제 패턴의 핵심 시사점

> **"문제와 지문이 해마다 바뀌어 출제될 뿐 해당 과목당 출제가 되어야 할 곳은 대충 정해졌다"**
> — 10년 기출 분석 결과

| 시사점 | 앱 기능으로의 전환 |
|--------|----------------|
| 출제 범위가 반복됨 | → **"이 단원에서 매년 O문제 출제"** 통계 기반 학습 우선순위 추천 |
| 작년 문제의 선지가 올해 정답으로 재출제 | → **선지 단위 학습** (문제 단위가 아님)이 핵심 |
